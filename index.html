GeoTools.

ระบบบริหารจัดการสมาชิก GeoTools เพื่อการเรียนรู้และแบ่งปัน สำหรับชาว QGIS ที่ต้องการพัฒนาทักษะไปด้วยกัน

เรื่องราวของ GeoTools

GeoTools เกิดขึ้นจากแนวคิดที่ต้องการรวมเครื่องมือการทำงานด้านแผนที่ (GIS) ให้เป็นหนึ่งเดียว เพื่อแก้ปัญหาความยุ่งยากในการดาวน์โหลดและติดตั้งปลั๊กอินหลายๆ ตัว โดยเรามุ่งเน้นการพัฒนาเครื่องมือที่ใช้งานง่าย มีประสิทธิภาพ และตอบโจทย์การทำงานจริงของช่างรังวัดและนักแผนที่ในประเทศไทย

นอกจากเครื่องมือที่ทรงพลังแล้ว เรายังสร้างระบบสมาชิกนี้ขึ้นมาเพื่อเป็นพื้นที่ในการเรียนรู้ แบ่งปันเทคนิค และเติบโตไปด้วยกันในสายงาน GIS โดยเงินสนับสนุนทั้งหมดจะถูกนำไปใช้เพื่อพัฒนาฟีเจอร์ใหม่ๆ ต่อไป

ดาวน์โหลดและติดตั้ง

ดาวน์โหลดปลั๊กอิน GeoTools และติดตั้งลงบน QGIS ของคุณ

download ตัวติดตั้งปลั๊กอิน (ZIP)

ดาวน์โหลดไฟล์ ZIP ของปลั๊กอิน GeoTools เวอร์ชันล่าสุด เพื่อนำไปติดตั้งในโปรแกรม QGIS

folder_zip ดาวน์โหลด GeoTools.zip

installs วิธีติดตั้งใน QGIS

  • 1. เปิดโปรแกรม QGIS และไปที่เมนู Plugins > Manage and Install Plugins...
  • 2. เลือกแท็บ Install from ZIP ด้านซ้ายมือ
  • 3. กดปุ่ม ... เพื่อเลือกไฟล์ `GeoTools.zip` ที่คุณดาวน์โหลดมา
  • 4. กดปุ่ม Install Plugin และรอจนระบบติดตั้งเสร็จสิ้น

วิธีติดตั้ง Web App บนมือถือ

เข้าใช้งานระบบแดชบอร์ดและคู่มือได้รวดเร็วขึ้นเหมือนมีแอปบนมือถือ

arrow_upward

สำหรับ iOS (Safari)

  1. เปิดเว็บไซต์นี้ด้วยเบราว์เซอร์ Safari
  2. กดปุ่ม แชร์ (Share) ที่แถบเมนูด้านล่าง (ไอคอนสี่เหลี่ยมลูกศรชี้ขึ้น)
  3. เลื่อนลงมาแล้วเลือก เพิ่มไปยังหน้าจอโฮม (Add to Home Screen)
  4. กด เพิ่ม (Add) ที่มุมขวาบน เป็นอันเสร็จสิ้น
more_vert

สำหรับ Android (Chrome)

  1. เปิดเว็บไซต์นี้ด้วยเบราว์เซอร์ Chrome
  2. กดปุ่ม เมนู (จุด 3 จุด) ที่มุมขวาบน
  3. เลือก ติดตั้งแอป (Install App) หรือ เพิ่มไปยังหน้าจอหลัก
  4. กด ติดตั้ง (Install) เพื่อยืนยัน
